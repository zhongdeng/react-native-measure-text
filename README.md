# react-native-measure-text

measure text size

## Installation

```sh
npm install react-native-measure-text
```

## Usage

```js
import MeasureText from '@zhongdeng/react-native-measure-text';

const style = {};

const props = {};

const { width, height } = await MeasureText.measureSingleText(
  'text',
  300,
  style,
  props
);
```
